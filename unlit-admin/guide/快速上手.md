## 环境准备
需要 Node.js 版本 >= 14.0.0
## 工具配置
IDE推荐使用vscode，并安装以下插件
- Vue Language Features (Volar)
- Vue 3 Snippets
- UnoCSS
- WindiCSS IntelliSense
- TSLint
- Prettier - Code formatter
- Beautify
## 拉取代码
``` sh
git clone https://gitee.com/cxice/unlit-admin-template.git
cd unlit-admin-template
```
## 安装依赖
``` sh
pnpm i
```
## 运行
``` sh
npm run dev
```
